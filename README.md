# TestFramework
测试框架

##  运行
运行入口：`src/test/case/testmain.py `
## 目录
1. config：`配置文件`
2. data：`测试数据`
3. drivers：`UI测试驱动`
4. logs：`日志`
5. report：`测试报告及缺陷截图`
6. src
    * test
        * case
            * API：`API接口类测试用例`
            * UI：`UI测试用例`
        * common：`封装浏览器的基本功能`
        * page：`PO模式，封装元素定位及操作`
        * suite：`测试集`
7. utils：`基础组件`
